# Changelog

## [0.1.0] - 2026-02-26

### Added

- 点击插件图标自动缓存剪切板内容，支持文本、图片、HTML 富文本
- 内容去重机制，相同内容不重复缓存
- Popup 缓存列表：缩略展示、相对时间、字符数统计
- 快捷复制，复制成功后显示"已复制 ✓"反馈
- 全屏查看：模态弹窗 + 新标签页全屏展示，支持图片、大段文本和 HTML 富文本
- 单条删除（二次确认）、一键清空
- 缓存上限可配置（默认 100 条，范围 10 ~ 999）
- 标签系统：为每条缓存添加 / 移除自定义标签
- 搜索过滤：按内容和标签关键词实时搜索（250ms 防抖），Ctrl/Cmd+F 快捷聚焦
- 滚动加载：默认 12 条，滚动到底部自动加载
- 统计信息：缓存数 / 上限、搜索匹配数、已占用空间
- 置顶 / 收藏：常用缓存置顶，带黄色左边框标识
- 快捷键唤起：默认 `Alt+Shift+C`，可在 `chrome://extensions/shortcuts` 自定义
- 导出 / 导入：JSON 格式一键导出全部记录，导入时自动去重并显示统计
- 暗色主题：支持 System / Light / Dark 三种模式，即时切换生效
- 国际化（中 / 英双语），即时切换生效
- 卡片按钮布局优化：复制按钮优先，删除按钮分隔隔开防止误触
- Popup 页头 GitHub 图标按钮
- 特殊页面降级处理
- Badge "✓" 缓存成功反馈
- Manifest V3，支持 Chrome 104+
- 纯前端实现，零运行时依赖，零网络请求
